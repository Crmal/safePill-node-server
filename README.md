# lionking

2022 명지대학교 캡스톤디자인2 safePill API Server

## How to start development server

```shell
yarn start:dev
```

## Commit conventions

- feat : 기능 추가
- refactor : 리팩토링, 기능은 그대로 두고 코드를 수정
- fix : 버그 수정
- style : 코드에 지장 안 가는 스타일적 코드 수정
- setting : eslint 변경, 라이브러리 설치 등
- docs : 주석 추가삭제, readme 변경 등
